Object-Oriented Programming (OOP)

OOP is a programming paradigm based on the concept of "objects", which can contain data and code: data in the form of fields (often known as attributes or properties), and code, in the form of procedures (often known as methods).

Key Concepts

  • Classes: Blueprints for creating objects.
  • Objects: Instances of classes.
  • Inheritance: Mechanism to create a new class using properties and methods of an existing class.
  • Encapsulation: Bundling data and methods that operate on that data within one unit.
  • Polymorphism: Ability to present the same interface for different underlying forms (data types).
